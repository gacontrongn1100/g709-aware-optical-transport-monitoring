# CV Positioning

## Project Title

G.709-Aware Optical Transport Monitoring Dashboard

## One-Line Description

Built a Python/Streamlit dashboard using a public optical transport dataset to analyze Pre-FEC BER, optical channel power and EDFA telemetry, with ITU-T G.709/OTN concepts used as the technical interpretation framework.

## CV Bullets

- Built a Python/Streamlit dashboard using the public Alibaba Cloud Transport System Dataset to analyze optical transport telemetry, including Pre-FEC BER, optical channel power, EDFA input/output power, gain, tilt and attenuation.
- Developed data-driven health classification rules to detect abnormal BER behavior, optical channel power drift and EDFA-related degradation without relying on private operator data.
- Applied ITU-T G.709/OTN concepts to interpret transport-channel quality, FEC-related degradation and DWDM/OTN monitoring workflows.
- Documented the data pipeline, KPI calculation, rule-based analysis logic, dataset limitations and project scope in a GitHub-ready technical report.
